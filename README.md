# MORE.TECH FUKIDID API

## HOW TO

### Local use

#### Requirements
* Python3.8
* Pipenv (`python3.8 -m pip install pipenv`)
* Make

#### Install dependencies
```shell
make install
```

#### Run
```shell
make dev
```

#### Test
```shell
make test
```
